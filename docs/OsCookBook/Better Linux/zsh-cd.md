 Zsh 历史路径显示与切换指南

  > [!note]
  > Ref: 本指南基于您当前系统中的 .zshrc 配置及 Oh My Zsh 插件生态。

  在您的当前配置中，已经启用了 z 插件和 Oh My Zsh。以下是几种最快捷的路径切换方法：

  1. 使用 z 插件（推荐）
    由于您的 plugins 中已包含 z，系统会自动记录您访问过的目录。

   - 跳转： 输入 z [部分路径名] 即可快速跳转。

   1   z imx  # 跳转到匹配 "imx" 的最常用路径
   - 查看历史权重： 直接输入 z 可列出所有记录的路径及其权重。

   1   z

  2. 使用目录栈 (Directory Stack)
    Oh My Zsh 默认开启了目录栈功能，非常适合在最近几个目录间切换。

   - 查看最近路径： 输入 d（dirs -v 的别名）查看带索引的历史列表。

   1   d
   2   # 输出示例：
   3   # 0  ~/imx/prj
   4   # 1  ~/SiFli-Suite
   5   # 2  /etc/nginx
   - 快速切换： 输入索引数字即可跳转。
     - 输入 1 跳转到列表中的第 1 个目录。
     - 输入 2 跳转到第 2 个，以此类推。

  3. 使用 cd 与补全
   - 最近两个目录切换： cd - 在当前目录和上一个目录间反复横跳。
   - 补全选择： 输入 cd - 后按下 Tab 键，zsh 会列出最近访问过的目录供您通过箭头选择。
