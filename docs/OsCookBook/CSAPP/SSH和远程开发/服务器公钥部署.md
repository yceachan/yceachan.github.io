## 部署公钥

```shell
# 创建.ssh目录（如果不存在）
mkdir -p ~/.ssh
# 设置目录权限（必须700，否则SSH会拒绝）
chmod 700 ~/.ssh
# 创建/编辑authorized_keys文件，粘贴本地复制的公钥; line by line per pub.
vi ~/.ssh/authorized_keys

chmod 600 ~/.ssh/authorized_keys
```

## 配置服务
```shell
cd /etc/ssh
sudo vim sshd_config
cd sshd_config.d
ls ...

sudo systemctl restart sshd
```
