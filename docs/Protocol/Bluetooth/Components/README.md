# GATT 服务组件笔记库 (Common GATT Services & Components)

这个笔记库旨在收集蓝牙协议中常用的基础 GATT 服务组件、描述符及其工作机制，包括但不限于：

- **CCCD** (Client Characteristic Configuration Descriptor)
- **DIS** (Device Information Service)
- **BAS** (Battery Service)
- **GATT 核心流程** (Notify, Indicate, Write without Response 等)
